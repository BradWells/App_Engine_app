My Projects

SIDS Monitoring Project - Senior Design

Currently, I am working on a project to help combat Sudden Infant Death Syndrome (SIDS). One of the possible symptoms of SIDS is abnormality of sleep states. Our team will capture data on infant sleep cycles and compile it together to get a baseline of an infants sleep patterns. From there, we will capture some nearly real-time data of an infant sleeping, and if we notice any irregularities, we will send an alert out.

Munchkin on Android

Another class project of mine was to bring the card game "Munchkin" to Android devices. The game would use Android phones to show the cards in a player's hand, and connect to a host tablet that would act as the game table. The game rules posed was quite a challenge to code, because they are very fluid, with some cards even allowing you to outright break the basic rules of the game. Even though we could not cover all of the cards that could be used in the game, we were able to get a simple working game created with several phones and a tablet.

Fluid Invitation Website
